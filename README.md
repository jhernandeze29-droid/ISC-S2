## 1.Sistema de cómputo 
Conjunto de hardware, software y usuarios que trabajan de forma integrada para procesar información y tener resultados.


## 2.Hardware
Conjunto de componentes físicos de una computadora, como el teclado, la pantalla, el mouse y la unidad central.


## 3.Software 
Conjunto de programas, aplicaciones y datos que dejan que una computadora o dispositivo funcione y realice varias tareas.


## 4.Usuario
Persona que interactúa con un sistema de cómputo y utiliza sus programas y recursos para hacer tareas.


## 5.Dato
Representación básica de un hecho, valor o información que por sí sola no tiene significado hasta ser procesada.


## 6.Entrada
Información o datos que entran en un sistema de cómputo para ser procesados.


## 7.Procesos
Conjunto de pasos o operaciones ordenadas que se hacen para lograr un objetivo o resultado específico.


## 8.Salida
Resultado de la información procesada por un sistema de cómputo, que se presenta a la persona.


## 9.Almacenamiento
Proceso y capacidad de guardar datos o información para su uso posterior en un sistema de cómputo.


## 10.Sistema operativo
Software principal que administra el hardware de la computadora y permite la ejecución de aplicaciones.



## 11.Archivo
Conjunto de datos almacenados en un dispositivo, identificado por un nombre y una extensión.


## 12.Base de datos
Conjunto organizado de datos relacionados entre sí, almacenados de forma estructurada para facilitar su acceso y gestión.


## 13.Computación en la nube
Uso de servicios informáticos a través de internet para almacenar, procesar y acceder a datos sin necesidad de equipos locales.


## 14.Inteligencia artificial (IA)
Rama de la informática que desarrolla sistemas capaces de imitar funciones humanas como el aprendizaje, el razonamiento y la toma de decisiones.


## 15.Ciberseguridad
Conjunto de medidas y prácticas hechas para proteger sistemas, redes y datos contra accesos no deseados y ataques digitales.
